## YouTube preview video

## Description
Simple 2D farming game-simulator. In the simulator, you have to grow plants, transform them and sell them.
## How to run
1. Clone repository
2. Open project in Unity Editor (2020.1.2f1)
3. Open "Game" scene
4. Press "Play" button
5. Have fun!
